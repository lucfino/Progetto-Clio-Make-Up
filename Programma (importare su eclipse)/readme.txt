Istruzioni per importare l'applicazione su eclipse 

1) Aprire Eclipse

2) Creare un nuovo progetto Java

3) Cliccare di destro sul progetto e selezione import

4) Scegliere Archive File e cliccare su Next

5) Selezionare il file jar da importare e cliccare su Finish

6) Nel proprio progetto ndare in src->lib, cliccare di destro sul file orm.jar e selezionare Build Path-> Add to Build Path

7) Inserire la cartella video nel progetto 

8) Importare il DB tramite il file video.sql ( dopo aver creato un database su localhost/phpmyadmin di nome clio )
